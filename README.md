# 動機

1. 目前無繁體中文平台、網路上的資源分散：</br>
   本身作為此繪畫 app 使用者，在尋找這類(筆刷及紙質參考)資源時，會感到有些麻煩，因為沒有繁中的平台集結，多是英文、簡體中文的平台；另外也有很多都是分享在個人的部落格、資源到處分散，會覺得看到眼花撩亂。

2. 對於其他網站的疑慮：</br>
   有很多時候用中文會找不到想要的筆刷，所以要改成打英文關鍵字才能找到，但找到又會擔心是詐騙網站，下載後電腦或平板可能會中毒或洩漏個資。
